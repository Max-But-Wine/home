# Cellar & Rind PWA

A mobile-friendly personal blog about **wine and cheese**.

## What it includes

- A home section introducing the blog.
- A latest posts section with curated tasting and pairing stories.
- A lightweight **Pairing Lab** to generate quick wine + cheese suggestions.
- Monthly cellar notes.
- A newsletter subscription form (saved in localStorage for demo purposes).
- Progressive Web App support:
  - web manifest,
  - service worker caching,
  - install prompt support.

## Run locally

This is a static HTML/CSS/JS site. No build step needed.

```bash
python3 -m http.server 4173
```

Open `http://localhost:4173`.
